MCP Remote Agent

Download and run the agent for your platform:

Windows (PowerShell)

Windows 10/11, Windows Server 2016+

irm https:///agent/windows | iex

Or with custom ID:

irm https:///agent/windows -OutFile agent.ps1; .\agent.ps1 -Id "MYPC"

Linux (Bash + Python)

Ubuntu, Debian, CentOS, Fedora, etc.

curl -s https:///agent/linux | bash

Or with custom ID:

curl -s https:///agent/linux -o agent.sh && bash agent.sh -i "MYSERVER"

macOS

macOS 10.15+ (Catalina and newer)

curl -s https:///agent/macos | bash

Or with custom ID:

curl -s https:///agent/macos -o agent.sh && bash agent.sh -i "MYMAC"

Android (Termux)

Termux app on Android

curl -s https:///agent/termux | bash

Or with custom ID:

curl -s https:///agent/termux -o agent.sh && bash agent.sh -i "MYPHONE"

Cross-Platform (Python)

Any platform with Python 3.7+

curl -s https:///agent/python -o agent.py && python3 agent.py

With options:

python3 agent.py --id "MYDEVICE" --server "ws://:3103"
